# نظام الإيجارات الزراعية (PWA)

تطبيق ويب تقدمي لإدارة الإيجارات الزراعية، المستأجرين، القطع، والمخالفات، باللغة العربية بالكامل (RTL).

راجع `docs/setup.md` لتعليمات النشر، و `PROJECT_PROMPT.md` لوصف المشروع الكامل.

## التشغيل محلياً
أي خادم ملفات ثابتة يكفي، مثال:
```
npx serve .
```
ثم افتح المتصفح على الرابط الذي يظهر.

⚠️ قبل التشغيل، عدّل `assets/js/config.js` وضع بيانات مشروع Supabase الخاص بك.
